# YAT Roadmap

This roadmap outlines the planned evolution of YAT - Yocto Automation Tool from its current development phase toward future releases. The focus is on establishing a reliable, practical, and product-ready experience for Yocto developers while keeping the roadmap realistic and implementation-oriented.

## v0.2-alpha (Current release)

### Foundation Release

**Goal:** Establish the core functionality of YAT and validate the tool on real Yocto development environments

**Status:** Completed

**Features:**
- Tool information and version reporting
- Yocto environment status checking
- Yocto path configuration management
- Environment diagnostics (Doctor)
- Build execution support
- Build dry-run functionality
- Recipe cleaning operations
- Build log collection and viewing
- Log analysis and troubleshooting support
- Project reporting
- Build history tracking
- Build statistics and analytics
- History filtering
- Statistics filtering
- Report export capability
- History export capability

## v0.3-alpha

### Stability & Real-world Validation

**Goal:** Improve reliability and validate YAT through real Yocto projects.

**Planned features:**
- Stability improvements and bug fixes
- Enhanced report formatting
- Improved log analysis output
- Better error handling and diagnostics
- Improved help and command documentation
- Daily usage validation on real Yocto projects
- Export format improvements
- Screenshots and sample output updates
- Multi-release Yocto compatibility validation
- Performance and usability enhancements

**Status:** Planned

## v0.5-beta

### Community Edition Beta Release

**Goal:** Prepare YAT Community Edition for wider evaluation and adoption.

**Planned features:**
- Community Edition features freeze
- Binary/demo package creation
- Installation and user guide
- Build comparison reporting
- Layer management support
- Layer listing and visibility tools
- Layer dependency validation
- Enhanced build analytics
- Configuration backup and restore
- Improved project reporting
- Community Edition evaluation package

**Status:** Planned

## v1.0

### Official Community Edition Release

**Goal:** Deliver a production-ready Community Edition.

**Planned features:**
- Stable and well-documented command interface
- Complete Community Edition release package
- AUTOREV detection
- Floating SRCREV detection
- Orphan bbappend detection
- Project audit capabilities
- Advanced export functionality
- Improved build history management
- Enhanced report generation
- Community Edition release documentation
- Production-readiness validation

**Status:** Planned
